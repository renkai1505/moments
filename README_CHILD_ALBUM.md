# 🌟 儿童相册系统 - Child Album

基于原有的朋友圈项目改造而成的专业儿童成长记录系统，让您可以轻松记录和管理孩子们的成长时光。

## ✨ 主要特性

### 👶 儿童档案管理
- **个人档案**: 创建详细的儿童档案，包含基本信息、身体数据等
- **成长追踪**: 自动计算年龄和出生天数
- **个性化**: 支持自定义头像和封面图片
- **完整信息**: 记录姓名、昵称、出生日期、性别、身高体重、血型、兴趣爱好等

### 📖 成长记录系统
- **多种类型**: 
  - 📏 成长发育记录
  - 🩺 健康记录  
  - 📚 学习记录
  - 🎮 游戏玩耍记录
  - 🏆 重要里程碑
- **丰富内容**: 文字、图片、位置、心情、天气等多维度记录
- **里程碑标记**: 突出显示重要的成长时刻
- **身高体重**: 专门追踪身体发育数据

### ⏰ 成长时间轴
- **时间线视图**: 按日期组织的美观时间轴
- **日期分组**: 同一天的记录智能分组显示
- **交互功能**: 查看详情、编辑记录等操作
- **视觉设计**: 清晰的视觉层次和儿童友好的界面

### 📊 统计分析 (即将推出)
- **成长统计**: 各类记录数量统计
- **发育曲线**: 身高体重增长趋势图表
- **里程碑回顾**: 重要时刻汇总展示

## 🚀 快速开始

### 环境要求
- **后端**: Go 1.23.3+
- **前端**: Node.js 18+, PNPM
- **数据库**: SQLite (自动创建)

### 安装和运行

#### 1. 克隆项目
```bash
git clone <项目地址>
cd moments
```

#### 2. 启动后端
```bash
cd backend
go mod tidy
go build -o dist/moments
./dist/moments
```

#### 3. 启动前端 (新终端)
```bash
cd front
npm install  # 或使用 pnpm install
npm run dev  # 或使用 pnpm dev
```

#### 4. 访问应用
打开浏览器访问: `http://localhost:3000`

## 📱 使用指南

### 1. 创建儿童档案
1. 登录系统后，点击首页的"创建儿童档案"按钮
2. 填写基本信息：姓名、昵称、性别、出生日期等
3. 可选择上传头像和封面图片
4. 添加身体信息、兴趣爱好等详细信息
5. 保存档案

### 2. 记录成长时光
1. 进入儿童档案详情页
2. 点击"添加记录"按钮
3. 选择记录类型（成长发育、健康、学习等）
4. 填写记录内容、上传图片
5. 可记录身高体重、位置、心情等信息
6. 标记重要里程碑事件

### 3. 浏览成长时间轴
1. 在儿童详情页选择"成长时间轴"
2. 按时间顺序查看所有成长记录
3. 点击任意记录查看详情或编辑

### 4. 管理多个儿童
- 支持为多个孩子创建独立档案
- 每个档案都有独立的成长记录
- 在首页可以切换查看不同孩子的信息

## 🚀 API 接口

### 儿童档案相关
```
POST   /api/child/save              # 保存儿童档案
GET    /api/child/list              # 获取儿童列表  
GET    /api/child/:id               # 获取儿童详情
DELETE /api/child/:id               # 删除儿童档案
```

### 成长记录相关
```
POST   /api/growth/save             # 保存成长记录
GET    /api/growth/list             # 获取成长记录列表
GET    /api/growth/timeline/:childId # 获取成长时间轴
GET    /api/growth/stats/:childId   # 获取成长统计
```

4. **里程碑标记**: 及时标记重要的里程碑事件
5. **定期备份**: 定期备份数据，确保回忆永不丢失
